# 楽曲形式

MIDI Sketch Bachは9つのバロック器楽形式で楽曲を生成します。各形式は固有の作曲規則に従い、異なる音楽構造を生み出します。

## オルガンシステム（形式 0--6）

バッハのオルガン・レパートリーの主要ジャンルをカバーする7つの形式。

### 0. フーガ

バロックポリフォニーの中核をなす純粋な対位法的フーガ。

**構造**: 単声部の主題提示で開始し、応答（通常5度上）が続きます。追加の声部が順次主題を提示します（提示部）。中間部ではエピソード（自由対位法）と新しい調での主題提示が交替します。ストレッタ（主題の重畳提示）で頂点に達し、主調での最終提示で締めくくられます。

- **デフォルト楽器**: オルガン
- **デフォルトBPM**: 85
- **デフォルト声部数**: 4
- **キャラクター**: 知的、構築的に厳密

::: info バッハのフーガ
バッハのオルガン・フーガ（BWV 578「小フーガ」ト短調、BWV 542など）はレパートリーの中でも最も高く評価される作品です。フーガはバロック形式の中で最も構造化された形式であり、声部がどのように入り、相互作用するかを規定する厳密な規則があります。
:::

```js
generator.generate({
  form: 'fugue',
  key: 7,        // G
  isMinor: true,
  numVoices: 4,
  character: 1
})
```

---

### 1. 前奏曲とフーガ

バッハの平均律クラヴィーア曲集やオルガン前奏曲をモデルとした、流麗な前奏曲と精緻なフーガのペア。

**構造**: 前奏曲は鍵盤楽器固有の音型 -- アルペジオパターン、ゼクエンツ、動機展開 -- により調性を確立します。続くフーガは提示部、エピソード、最終提示を持つ完全な対位法作品です。

- **デフォルト楽器**: オルガン
- **デフォルトBPM**: 90
- **デフォルト声部数**: 4
- **キャラクター**: バランス型、叙情的な前奏曲と知的なフーガ

::: info バッハの前奏曲とフーガ
前奏曲とフーガの組み合わせはバッハの代名詞的形式であり、平均律クラヴィーア曲集（BWV 846--893）と偉大なオルガン前奏曲とフーガ（BWV 532, 541, 548）の両方に代表されます。
:::

```js
generator.generate({
  form: 'prelude-and-fugue',
  key: 0,        // C
  isMinor: false,
  numVoices: 4
})
```

---

### 2. トリオ・ソナタ

バッハのオルガン・トリオ・ソナタをモデルとした3声部テクスチュア。2つの上声部が独立したバスライン上で相互に作用します。

**構造**: 2つの上声部は模倣対位法で旋律的アイデアを交換しながら独立性を維持します。バス声部（通常オルガンのペダルで演奏）は独自の旋律的性格を持つ基盤を提供します。スケール設定に応じて複数楽章が生成されることがあります。

- **デフォルト楽器**: オルガン
- **デフォルトBPM**: 90
- **デフォルト声部数**: 3
- **キャラクター**: 対話的、2つの「高音」声部の掛け合い

::: info バッハのトリオ・ソナタ
バッハの6つのオルガン・トリオ・ソナタ（BWV 525--530）は、オルガン・レパートリーの中でも最も技術的に難しい作品であり、両手と両足の完全な独立性を要求します。各「声部」は独自の鍵盤またはペダルを持ちます。
:::

```js
generator.generate({
  form: 'trio-sonata',
  key: 5,        // F
  isMinor: false,
  bpm: 90
})
```

---

### 3. コラール前奏曲

讃美歌旋律（コラール）に対位法的伴奏を施した楽曲。

**構造**: 定旋律（コラール旋律）が1つの声部（通常はソプラノ）に長い音符で現れます。他の声部はコラールを装飾し和声化する独立した対位法的ラインを織りなします。瞑想的で豊かなテクスチュアの楽曲となります。

- **デフォルト楽器**: オルガン
- **デフォルトBPM**: 72
- **デフォルト声部数**: 4
- **キャラクター**: 敬虔的、瞑想的、豊かな層構造

::: info バッハのコラール前奏曲
バッハは生涯を通じて150以上のコラール前奏曲を作曲しました。オルゲルビュッヒライン（BWV 599--644）から「ライプツィヒ」コラール（BWV 651--668）まで、コラール前奏曲の伝統はルター派の典礼の中心でした。
:::

```js
generator.generate({
  form: 'chorale-prelude',
  key: 9,        // A
  isMinor: false,
  numVoices: 4,
  bpm: 66
})
```

---

### 4. トッカータとフーガ

自由形式のトッカータと厳格なフーガを組み合わせた劇的な二部構成の作品。

**構造**: トッカータセクションはヴィルトゥオーゾ的なパッセージ、劇的な音階、持続和音で開始します。その後、すべての声部での主題提示、エピソード、クライマックスを持つ完全に展開されたフーガへと移行します。

- **デフォルト楽器**: オルガン
- **デフォルトBPM**: 100
- **デフォルト声部数**: 4
- **キャラクター**: 壮大、劇的、ヴィルトゥオーゾ的

::: info バッハのトッカータとフーガ
ニ短調のトッカータとフーガ（BWV 565）はバッハの最も有名なオルガン作品です。トッカータ（イタリア語の「toccare（触れる）」に由来）は、構造的なフーガが始まる前に自由な走句で演奏者の技巧を披露します。
:::

```js
generator.generate({
  form: 'toccata-and-fugue',
  key: 2,        // D
  isMinor: true,
  numVoices: 4,
  bpm: 80
})
```

---

### 5. パッサカリア

繰り返されるバス主題（オスティナート）上に構築される変奏形式。

**構造**: バス主題の提示（通常4小節または8小節）で開始します。連続する変奏が、繰り返されるバス上に新しい旋律的・リズム的素材を重ね、複雑さが増していきます。テクスチュアは単声部から完全なポリフォニーへと厚みを増します。バス主題に基づくフーガで締めくくられることもあります。

- **デフォルト楽器**: オルガン
- **デフォルトBPM**: 76
- **デフォルト声部数**: 4
- **キャラクター**: 高まる緊張感、記念碑的

::: info バッハのパッサカリア
バッハのパッサカリアとフーガ ハ短調（BWV 582）は変奏形式の傑作であり、8小節のバス主題上に20の変奏を構築した後、二重フーガで頂点を迎えます。この形式は17世紀のスペインとイタリアに起源を持ちます。
:::

```js
generator.generate({
  form: 'passacaglia',
  key: 0,        // C
  isMinor: true,
  numVoices: 4,
  scale: 'long'
})
```

---

### 6. 幻想曲とフーガ

自由な幻想曲と構造的なフーガの組み合わせ。

**構造**: 幻想曲セクションは即興的でラプソディックな書法 -- 大胆な和声探求、予期しない転調、厳格な形式に縛られない劇的なジェスチャー -- を特徴とします。これに続いて完全に展開されたフーガが対位法的厳密さと形式的完結をもたらします。

- **デフォルト楽器**: オルガン
- **デフォルトBPM**: 88
- **デフォルト声部数**: 4
- **キャラクター**: 想像力豊かな幻想曲、規律あるフーガ

::: info バッハの幻想曲とフーガ
バッハのオルガン幻想曲とフーガ（BWV 537, 542, 561）は、幻想曲（イタリア語の「fantasia（想像力）」に由来）の即興的自由さとフーガの知的規律を組み合わせています。ト短調の幻想曲とフーガ（BWV 542）は、史上最も偉大なオルガン作品の一つとされています。
:::

```js
generator.generate({
  form: 'fantasia-and-fugue',
  key: 7,        // G
  isMinor: true,
  numVoices: 4,
  bpm: 88
})
```

## 独奏楽器システム（形式 7--8）

無伴奏弦楽器のための2つの形式。

### 7. チェロ前奏曲

バッハの無伴奏チェロ組曲の冒頭楽章をモデルとした、独奏チェロのための流麗な前奏曲。

**構造**: 単一の旋律線を通じて和声進行を描き出す連続的なアルペジオ的音型。チェロの音域の変化から複数の暗示された声部が浮かび上がります -- 低音、内声の和声、上声の旋律が一つの楽器の能力の中で織り交ぜられます。

- **デフォルト楽器**: チェロ
- **デフォルトBPM**: 80
- **デフォルト声部数**: 3
- **キャラクター**: 流麗、和声的に豊か、瞑想的

::: info バッハの無伴奏チェロ組曲
6つの無伴奏チェロ組曲（BWV 1007--1012）はチェロ・レパートリーの礎石です。各組曲はアルペジオパターンを通じて調性を確立する前奏曲で始まり、単線楽器上に複数の声部の幻想を作り出します。
:::

```js
generator.generate({
  form: 'cello-prelude',
  key: 7,        // G
  isMinor: false,
  instrument: 'cello'
})
```

---

### 8. シャコンヌ

バッハのパルティータ第2番ニ短調のシャコンヌに着想を得た、独奏ヴァイオリンのための壮大な変奏形式。

**構造**: 繰り返される和声進行（通常4小節または8小節）の上に、楽器のあらゆる表現の可能性を探求する一連の変奏として展開されます。複数の暗示された声部 -- 和音的書法、アルペジオ的パッセージ、単線の旋律 -- が独奏楽器上に交響的なテクスチュアを作り出します。

- **デフォルト楽器**: ヴァイオリン
- **デフォルトBPM**: 76
- **デフォルト声部数**: 3
- **キャラクター**: 叙事詩的、強烈に表現的、建築的に壮大

::: info バッハのシャコンヌ
パルティータ第2番ニ短調（BWV 1004）のシャコンヌは、音楽全体の中でも最も偉大な業績の一つと広くみなされています。約15分の中で、バッハは単純な和声的枠組みから一つの世界を構築し、ヴァイオリンに可能なあらゆる音域と技法を探求しています。
:::

```js
generator.generate({
  form: 'chaconne',
  key: 2,        // D
  isMinor: true,
  instrument: 'violin'
})
```

## 用途別の形式選択ガイド

| 用途 | 推奨形式 |
|------|---------|
| 壮大なオルガン作品 | トッカータとフーガ、幻想曲とフーガ、前奏曲とフーガ |
| 対位法の学習 | フーガ、トリオ・ソナタ |
| 独奏チェロ作品 | チェロ前奏曲 |
| 独奏ヴァイオリン作品 | シャコンヌ |
| 瞑想的・敬虔的楽曲 | コラール前奏曲 |
| 変奏技法 | パッサカリア、シャコンヌ |
| 室内楽的テクスチュア | トリオ・ソナタ |
| 手軽な生成 | フーガ（3声でコンパクト） |
| 複雑なポリフォニー | パッサカリア、フーガ（4声） |
| 劇的・ヴィルトゥオーゾ的作品 | トッカータとフーガ、幻想曲とフーガ |

::: tip
各形式の完全なデフォルト値と有効範囲については[プリセットリファレンス](/ja/docs/presets)をご覧ください。
:::
